# Wu Boshi Geometry-First Conic Superfamily

This file is the consolidated superfamily surface for the geometry-first conic line.

Its purpose is:

- unify geometry-first conic routes that are currently spread across many smaller modules
- separate recognition, focal-geometry, direct readout, and final-seal roles

## Core Principle

Do not default to coordinates just because the object is a conic.

Ask first:

- what geometric controller is already visible?
- what hidden whole should be completed?
- what lower-dimensional or triangular object now owns the target?
