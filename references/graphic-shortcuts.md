# Wu Boshi Graphic Shortcuts

This file defines the move:

- if the figure already exposes the route, let the figure do the work before symbolic machinery

## Core Principle

A good diagram can collapse:

- several equations into one picture
- several cases into one spatial relation
- several formal steps into one visible constraint

## Common Uses

- conics
- function graph reading
- symmetric geometry
- distance and area relations
- axis / midpoint / support-line problems

## Guiding Rule

Prefer the figure first when it can:

- eliminate impossible branches
- reveal a controlling triangle
- expose a midpoint, axis, or center
- show the answer is to be read, not rebuilt

## Hard Rule

Graphic shortcut is valid only if the picture preserves the decision structure.
If the picture only decorates the algebra, it is not a shortcut.
