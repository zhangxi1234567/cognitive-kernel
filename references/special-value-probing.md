# Wu Boshi Special-Value Probing

This file defines the move:

- use specially chosen values, cases, positions, or representatives to expose the controlling structure quickly

## Core Principle

Special values are not random plugs.

They are chosen because they:

- collapse the algebra
- separate neighboring candidates
- expose symmetry or imbalance
- reveal a hidden control variable
- pressure-test a guessed route cheaply

## What Counts As A Good Special Value

A good special value should do at least one of these:

- make many terms vanish
- make the sign obvious
- make the comparison axis visible
- hit a boundary, midpoint, or symmetry seam
- turn an abstract object into a canonical representative

## Common Uses

- test landmark parameter values
- test boundary or midpoint cases
- choose convenient coordinates or representatives
- expose hidden equalities or monotone behavior
- turn a complicated expression into a visible mother form

## Difference From Blind Plugging

Blind plugging asks:

- what easy number can I try?

Wu Boshi special-value probing asks:

- what value is most likely to expose the controlling seam?

## Where It Transfers

- arithmetic and algebra
- function and derivative problems
- geometry via convenient coordinates
- abstract algebra via representatives
- debugging via smallest failing input

## Hard Rule

If the chosen value is easy but not structurally dangerous, it is not a true special-value move.
