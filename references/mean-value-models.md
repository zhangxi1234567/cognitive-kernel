# Wu Boshi Mean-Value Models

This file defines the move:

- replace a full derivative flow with one bridge quantity, midpoint relation, or average-style control object

## Core Principle

Many “derivative” problems are not really about derivative mechanics.
They are about:

- one bridge quantity
- one mean-value relation
- one secant-to-tangent compression

## Common Uses

- derivative inequality
- parameter monotonicity
- midpoint / secant / tangent bridges
- function comparison through one average model
- motion-style bridge objects in math-physics transfer

## Math-Physics Note

These models often become lower-floor when read as:

- average speed
- bridge displacement
- secant-to-tangent motion comparison

That is not merely pedagogy.
It often exposes the real controlling object faster than raw derivative symbols.

## Hard Rule

If the mean-value object does not reduce the number of moving parts, it is not a real mean-value model.
