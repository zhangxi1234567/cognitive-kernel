# Wu Boshi Answer Shape Lock

This file defines the reduction move:

`use the shape of the answer to search backward`

## Core Principle

Many exam answers are not arbitrary.

They often want:

- integer
- simple fraction
- radical
- fixed constant
- fixed line / fixed point
- simple order relation

Instead of ignoring that, use it.

## Trigger

Use this move when:

- the answer is multiple choice
- the problem asks for a minimum / maximum
- the result looks likely to be clean
- the target is “prove fixed / constant / line / point”

## Fast Move

1. Guess the answer shape, not necessarily the exact answer.
2. Ask what structure would force such a shape.
3. Search for the mechanism that naturally produces it.

## Why It Works

The answer’s shape often reflects the underlying controlling structure.

## Where It Transfers

- parameter problems
- geometry fixed-object problems
- ordering and comparison questions
- physics threshold values
- chemistry limiting ratios

## Failure Mode

- answer worship without mechanism
- too many equally pretty candidates
- the clean-looking value is only cosmetic
