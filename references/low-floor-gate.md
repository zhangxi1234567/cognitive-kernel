# Wu Boshi Low-Floor Gate

This file defines the gate:

- a route only counts as true dimensionality reduction if a lower-complexity solver can actually carry the main route

## Core Principle

It is not enough that the explanation starts with a smaller intuition.

The reduced route must satisfy all three:

1. the main solve skeleton changed
2. the controlling structure became smaller
3. a lower-complexity solver could continue the route without hidden expert imports

If condition 3 fails, the answer only front-loaded simplification and then climbed back too early.

## What Counts As Passing

A route passes the low-floor gate when:

- the key step can be executed with earlier-layer primitives
- the decisive move is not secretly delegated to a high-formalism formula
- the explanation does not require the learner to accept an imported object they could not rebuild

## What Fails

A route fails when:

- the first half sounds elementary, but the hinge still needs a chapter-default formal device
- the answer says “now just set up slope / discriminant / derivative / vector identity / matrix form” without first lowering those objects
- the lower-complexity layer only serves as motivation, not execution

## Low-Floor Questions

Ask:

1. Could a lower-level solver perform the decisive hinge, not just admire it?
2. Did I smuggle in a formula or object that belongs to the old harder route?
3. If the formal notation were removed, would the reduced route still move?
4. Is the final seal minimal enough that the lower-floor route still owns the solution?

## Hard Rule

Do not claim “middle-school can do this high-school problem” unless the middle-school layer can actually carry the decisive hinge.

Do not claim “high-school can do this university problem” unless the high-school layer still owns the route after the shell dies.
