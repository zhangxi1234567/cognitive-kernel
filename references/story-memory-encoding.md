# Wu Boshi Story And Memory Encoding

This file defines the move:

- convert a method family into a memorable world without changing its control mechanism

## Core Principle

Story, cartoon, animation, and game packaging are useful when they improve:

- recognition
- retention
- route recall

They fail when they replace mechanism with vibe.

## Good Encoding

A good memory encoding should preserve:

- what to notice
- what move to make
- what condition licenses the move

## Common Uses

- prototype banks such as seven triangles
- mother-pattern dictionaries
- fast comparison families
- repeated route maps that are hard to retain in dry form

## Verification Rule

After encoding, the learner should still be able to answer:

- what is the primitive?
- what triggers it?
- why is it legal?

If not, the encoding became decoration instead of method support.
