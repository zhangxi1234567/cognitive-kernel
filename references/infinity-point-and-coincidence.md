# Wu Boshi Infinity-Point And Coincidence

This file defines the move:

- use coincidence cases and “points at infinity” as lawful simplifying limits when they preserve the controlling relation

## Core Principle

Many geometry and conic problems simplify most at:

- tangent/coincidence
- repeated intersection
- asymptotic/infinite direction

These are not illegitimate hacks.
They are structural edge cases.

## Common Uses

- conic fixed-point problems
- slope product or slope sum problems
- line-family questions
- limiting range arguments

## Fast Move

1. identify the easiest coincidence case
2. identify the asymptotic / infinite-direction case if relevant
3. compute the two simplest edge configurations
4. use them to guess or lock the target structure

## Hard Rule

Do not use infinity-point language as empty rhetoric.
You must say what relation survives in the limit.
