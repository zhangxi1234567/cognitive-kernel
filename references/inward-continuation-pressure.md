# Inward Continuation Pressure

This file defines a weak compatibility pressure:

- once a route has landed on a serious carrier, it may still feel naturally unfinished from the inside

Its purpose is:

- make continuous reduction feel more self-propelled
- reduce dependence on externally imposed “go deeper” pressure
- preserve continuity inside one route without turning it into a descent program

## Core Claim

Some of the strongest routes do not need to be told:

- continue
- reduce again
- ask another why

They continue because the current carrier itself still feels:

- too thick
- not yet target-complete
- not yet at the lowest honest layer

The package should preserve that kind of inward unfinishedness.

## What This Pressure Means

Inward continuation pressure means only:

- the current route can still become smaller from within its own carrier

It does not mean:

- now recurse
- now run another layer
- now perform one fixed next move

Those remain model-side.

## What The Package May Preserve

The package may preserve:

- `inward_continuation_pressure`

with values like:

- `low`
- `medium`
- `high`

This is:

- weak inner-route continuity pressure

Not:

- a continuation command
- a recursive ladder
- a visible self-question script

## Relation To Existing Pressures

This pressure often grows when:

- `middle_layer_nonfinality` is high
- `lowest_honest_layer_attraction` is high
- `deeper_carrier_coherence` is high
- the current carrier is not yet target-complete

It is not identical to any one of them.

Those pressures say:

- the route should not fully settle

This pressure adds:

- the route may continue inward on its own current carrier rather than by external restart or visible prompting
