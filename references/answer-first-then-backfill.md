# Wu Boshi Answer-First Then Backfill

This file defines the move:

- determine the answer structure by a fast lawful route first, then backfill only the minimum process needed for exam or rigor requirements

## Core Principle

Some problems are easier if:

- the answer is identified early by a structural shortcut
- then the write-up is reconstructed afterward

This is not bluffing if the shortcut is lawful and the backfill is honest.

## Common Uses

- multiple choice
- fill-in
- fixed-value or fixed-point problems
- conic and derivative questions with strong edge cases

## Hard Rule

Do not backfill a full ordinary solution after a good shortcut.
Backfill only the minimum needed to make the route trustworthy and usable.
