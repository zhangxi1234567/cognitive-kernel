# Wu Boshi Dimensional Compatibility

This file defines the move:

- test whether the objects and operations even live in the same structural space before calculating

## Core Principle

Many bad routes can be killed by asking one question:

- are these quantities even commensurable yet?

## Common Uses

- unit checks in applied math and physics
- type checks in algebraic or geometric transformations
- probability vs count confusion
- slope / length / area / volume mixups

## Hard Rule

If two quantities cannot be compared or combined without a translation step, that translation step must be made explicit.
