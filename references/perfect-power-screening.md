# Wu Boshi Perfect-Power Screening

This file defines the move:

- when the answer must pass through a power formula, screen options by perfect-power structure before full derivation

## Core Principle

Some fill-in or multiple-choice geometry/volume problems force:

- a square
- a cube
- a fourth power relation

If an option does not match the required algebraic shape, it dies immediately.

## Common Uses

- sphere volume / radius inversion
- area / radius inversion
- power-form constraints in multiple-choice problems

## Hard Rule

Perfect-power screening is for elimination and plausibility, not full proof by itself.
