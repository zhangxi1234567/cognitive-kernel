# Reflex Surfaces

This file defines the smallest descriptive reflex layer.

Its purpose is:

- preserve a tiny amount of descriptive runtime residue
- prevent that residue from becoming a control plane

## Allowed Surface

Only descriptive residue is allowed, such as:

- ordinary flow still feels too thick
- a smaller carrier now feels more trustworthy
- the seal still feels insufficient
- stopping now feels honest

## Forbidden Surface

Do not expose:

- activation order
- stage gates
- density tables
- continuation programs
- route selectors
- explicit plurality dashboards

## Final Rule

A reflex surface is acceptable only if it cannot tell the host what to do next.
