# Wu Boshi Estimation Aids

This file defines the move:

- use remembered anchor values, rough scales, and estimation tools to separate answers before full derivation

## Core Principle

Exact calculation is often unnecessary when:

- the answer choices are far apart
- one rough scale already kills most branches
- one remembered constant is enough to rank the options

## Common Tools

- remembered log values such as `ln 2`, `ln 3`, `ln 5`, `ln 7`
- rough square-root benchmarks
- ratio magnitude checks
- sign and size around `0` and `1`
- simple physical scales and order-of-magnitude sense

## Good Use

Use estimation to:

- guide route choice
- eliminate impossible options
- support a visual or comparison shortcut

## Hard Rule

Estimation is a separator first, not a proof substitute.
If the problem needs exact certification, add the smallest honest seal after the estimate.
