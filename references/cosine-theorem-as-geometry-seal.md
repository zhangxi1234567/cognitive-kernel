# Wu Boshi Cosine Theorem As Geometry Seal

This file defines the move:

- when geometry has already been compressed to one non-right triangle, use the cosine theorem as the shortest reliable seal

## Core Principle

The cosine theorem is often not the main route.
It is the minimal final seal after:

- picture-first reduction
- focal triangle completion
- side relation compression

## Trigger

Use it when:

- the controlling triangle is non-right
- geometry has already been reduced to a triangle-side relation
- brute coordinates would be longer

## Hard Rule

Do not start with the cosine theorem.
Use it only after the controlling triangle is already visible.
