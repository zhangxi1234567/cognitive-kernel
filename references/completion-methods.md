# Wu Boshi Completion Methods

This file defines the move:

- when a geometric object is incomplete, damaged, or partial, restore the full simpler whole before solving

## Core Principle

Some hard geometry is hard only because the object is incomplete.

Fast route:

- complete it
- solve the full simple object
- map back to the partial original

## Common Uses

- truncated solids
- half-objects
- partial sections
- missing symmetric parts
- one-focus views that really belong to a two-focus whole

## Hard Rule

Completion is valid only if the restored whole preserves the target-relevant structure.
