# Wu Boshi Focus Completion

This file defines the move:

- when a conic problem gives one focus, one chord, or one partial focal relation, complete the missing focus structure before computing

## Core Principle

Many conic problems look one-sided only because one focus or one side of the structure is named explicitly.

The decisive geometry often appears only after:

- completing the second focus
- drawing the full focal triangle
- restoring the hidden rectangle / midpoint structure

## Common Uses

- ellipse focal triangles
- chord problems through the origin
- midpoint / diagonal relations
- symmetry completion in focal geometry

## Hard Rule

If the picture names only one focus but the geometry is really two-focus controlled, do not stay in the one-focus view.
