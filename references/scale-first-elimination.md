# Wu Boshi Scale-First Elimination

This file defines the move:

- reject candidates early when their magnitude is structurally impossible

## Core Principle

Before proving a route, ask:

- is this answer even in the right size universe?

This is especially strong when:

- options are far apart
- the picture gives a visible maximum/minimum scale
- growth order clearly dominates

## Common Uses

- multiple choice elimination
- conic and geometry estimates
- derivative / asymptotic comparison
- probability bounds
- physical scale screening

## Hard Rule

Scale-first elimination is strongest when the route already exposes the controlling capacity or bound.
Without that capacity, it becomes hand-waving.
