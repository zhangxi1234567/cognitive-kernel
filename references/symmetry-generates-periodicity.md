# Wu Boshi Symmetry Generates Periodicity

This file defines the deeper law behind many symmetry-to-periodicity routes.

Its purpose is:

- explain why axis symmetry and center symmetry can force periodic structure
- treat periodicity as generated by repeating symmetry operations

## Core Rule

When a function or structure has:

- a symmetry axis
- a symmetry center
- or multiple parallel symmetry features

ask:

- what translation is produced by composing these symmetries?

Then compress:

- symmetry data -> period

## Bottom-Layer Logic

- one reflection changes orientation
- two compatible reflections produce a translation
- alternating axis/center symmetries create repeated copies of the same structure

So periodicity appears because:

- local symmetry operations compose into global repetition

## What This Explains

- axis spacing -> half-period
- center spacing -> half-period
- axis + center -> quarter/half-period relations
- odd/even plus shifted symmetry -> generated periodicity

## Trigger Signals

- `f(a-x)=f(b+x)`
- center-symmetry conditions
- odd/even plus shifted conditions
- repeated mirror structure in graphs

## Failure Boundary

Do not overfire this move when:

- the symmetry is only local
- the domain blocks global continuation
- the generated translation does not preserve the whole structure

## Transfer

- school math: function periodicity and graph repetition
- university math: group action viewpoint, generated translations
- cross-domain: repeated local invariant operations creating a global cycle
