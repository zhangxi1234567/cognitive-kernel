# Wu Boshi When The Standard Route Breaks

This file defines the move:

- explicitly detect when the standard route has become structurally blocked, and switch methods instead of grinding harder

## Core Principle

Ordinary solving often responds to blockage with more algebra.

Wu Boshi style should respond by asking:

- what exactly is blocked?
- what new representation would unblock it?

## Typical Block Signs

- the target is asymmetric but the available machinery is symmetric
- the introduced variables are multiplying rather than simplifying
- Vieta gives the wrong shape of information
- coordinates grow but the target still hides

## Fast Move

1. Name the obstruction.
2. Name the current machinery that cannot control it.
3. Switch to a target rewrite, symmetry rewrite, aggregate rewrite, or geometric picture that can.

## Hard Rule

Grinding harder on a blocked route is not perseverance.
It is method failure.
