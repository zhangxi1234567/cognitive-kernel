# Wu Boshi Geometry-First Principle

This file defines the move:

- when the real control is geometric, let the figure speak before the algebra

## Core Principle

If a problem can be compressed by:

- one diagram
- one triangle
- one midpoint relation
- one rectangle property
- one visual ratio

then geometry should come first.

## Common Signals

- focal distances
- midpoint / diagonal relations
- symmetry lines
- tangency and support lines
- ratio conditions that look cleaner in the picture than in coordinates

## Hard Rule

Do not translate immediately into coordinates when the figure already exposes:

- the controlling shape
- the decisive ratio
- the shortest reliable route
