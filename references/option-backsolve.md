# Wu Boshi Option Backsolve

This file defines the move:

- use the options or answer format to reverse-check or reverse-construct the route quickly

## Core Principle

When the problem is multiple-choice or fill-in with a clean answer, options are not only outputs.
They are extra structure.

They may help with:

- elimination
- scale separation
- route selection
- reverse validation

## Good Uses

- options are far apart
- one candidate answer implies a much smaller structure
- a guessed route can be checked against one option cheaply

## Hard Rule

Do not let options replace mechanism.
Use them to steer or validate the route, not as a substitute for the route.
