# Wu Boshi Logic Laws Of Dimensional Reduction

This file defines the deepest universal laws behind the library.

It sits beneath:

- visible methods
- primitives
- clusters
- even cognitive moves

## Why This Layer Exists

A good reduction system should answer not only:

- what move should I use?

but also:

- why does this move work so often?

That “why” is the logic-law layer.
