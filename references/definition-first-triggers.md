# Wu Boshi Definition-First Triggers

This file defines the move:

- when a problem gives a direct defining relation, use the definition before auxiliary machinery

## Core Principle

A defining relation is often the shortest route.

Examples:

- conic focus/directrix distance
- probability as favorable over total under a valid uniform encoding
- periodicity from the defining shift relation
- abstract function properties from the functional equation itself

## Trigger

Use this when:

- the problem statement already contains the defining relation
- standard methods would rebuild what the definition already gives
- the target is directly linked to the definition picture

## Fast Move

1. Write the definition in the problem's own variables.
2. Ask what it already controls.
3. Delay all secondary machinery until the definition has spent its value.

## Hard Rule

Do not jump to coordinates, derivatives, or long algebra when the definition already names the shortest bridge.
