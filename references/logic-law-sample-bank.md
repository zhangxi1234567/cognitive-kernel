# Wu Boshi Logic-Law Sample Bank

This file collects concrete sample problems under the deeper logic laws they test.

Purpose:

- connect the abstract law layer to real tasks
- keep regression from floating as pure theory
- show transfer across levels and topics

## How To Use

For each law:

- start from one easy sample
- add one exam-like sample
- add one advanced sample when available

The point is to test the law across surfaces, not to worship one famous problem.
