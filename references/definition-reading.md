# Wu Boshi Definition Reading

This file defines the move:

- treat a new-definition problem as a reading-and-translation problem before treating it as a calculation problem

## Core Principle

In many new-definition problems, the first failure is not algebra.
It is reading.

So the right order is:

1. parse the definition
2. identify the actors, operations, and constraints
3. test with tiny examples
4. only then compute

## Practical Reading Questions

- what is the object?
- what is being done to it?
- what are the conditions or restrictions?
- what counts as a legal instance?
- what tiny example would confirm I understood it?

## Hard Rule

If the definition is not yet understood structurally, any further calculation is fake progress.
