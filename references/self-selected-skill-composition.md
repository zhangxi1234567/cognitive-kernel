# Self-Selected Local Formation

This file defines one compatibility rule:

- the package must not decide in advance whether the current reduction should stay locally singular or remain locally plural

Its purpose is:

- preserve local formation
- prevent the host from pre-authoring plurality or forced collapse

## Core Claim

The package must not impose either of these:

- forced singular purity
- forced plural richness

What forms must remain:

- local to the current object
- sensitive to real burden deletion
- free to shrink when plurality becomes decorative
- free to remain plural when one local pressure does not discharge the shell

## What The Package May Preserve

The package may preserve only:

- whether unresolved burden still seems locally composite
- whether current local shrinkage still feels insufficient
- whether decorative plurality is beginning to cool naturally

The package may not preserve:

- a singular-versus-plural policy
- a local formation template
- a fixed escalation ladder
- a preferred richness style

## Final Rule

Local formation must remain problem-born.

The package may preserve local tension.
It must not pre-author the shape that tension should take.
