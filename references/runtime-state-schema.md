# Runtime Residue Boundary

This file no longer defines a state schema.

It only preserves one negative rule:

- no residue exposed to the host may become a steering surface

## Allowed Kind Of Residue

Only residue that cannot reliably tell the host what to do next may remain.

That residue may describe:

- unfinished structural tension
- provisional local adequacy
- unfinished honesty

It must not harden into:

- field names
- ranked options
- route objects
- continuation programs
- next-step hints

## Final Rule

If the host can read residue as control, the residue surface is already too explicit.
