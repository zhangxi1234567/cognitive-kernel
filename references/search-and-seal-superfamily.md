# Wu Boshi Search-And-Seal Superfamily

This file is the consolidated superfamily surface for the guess / probe / verification line.

Its purpose is:

- unify route guessing, answer-shape pressure, witness probing, special-value probing, and minimal sealing
- show how a solver can search smaller and certify cheaper without collapsing into bluff or ordinary derivation

## Core Principle

A good solver does not always begin with full proof search.

Often the stronger route is:

1. shrink the answer space
2. probe the weakest seam
3. seal only the decisive hinge

That is the search-and-seal family.

## The Three Internal Layers

### 1. Search Layer

Use:

- `Guess Then Verify`
- `Answer-Shape Pressure`
- `Baited-Route Recognition`

Purpose:

- choose the likely smaller route before spending the full proof bill

### 2. Probe Layer

Use:

- `Witness Probing`
- `Special-Value Probing`
- `Puzzle Assignment`

Purpose:

- test the most revealing seam
- expose structure cheaply
- kill wrong branches before they grow

### 3. Seal Layer

Use:

- `Rapid Verification`
- `Minimal exact check`
- `Trust bill`

Purpose:

- lock the shortcut with the smallest honest certification

## Universal-Age Reading

This superfamily is age-agnostic.

It may look like:

- child: try an easy case and see what breaks
- teenager: test boundary / symmetry / special values
- undergraduate: probe a witness and seal with one formal hinge
- graduate / research: conjecture a mechanism, attack the weakest seam, and certify only the bottleneck

The underlying structure stays the same:

- search smaller
- probe cheaper
- seal only what matters

## What This Superfamily Already Knows

- how to use clean answer shapes without hallucinating
- how to choose revealing values instead of random easy values
- how to stop proof bills from regrowing after a good route is found

## What Still Depends On More Source Access

- more transcript-level public evidence for stable guess subtypes
- stronger mapping from public `先猜后证` wrappers into finer route subfamilies

## Hard Rule

Do not confuse:

- “guessing”

with:

- “bluffing”

This superfamily is only valid when probing and sealing are real, not theatrical.
