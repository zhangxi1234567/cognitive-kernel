# Wu Boshi Eccentricity Direct Readouts

This file defines the move:

- read eccentricity directly from a controlling ratio or special triangle once the conic geometry has been compressed enough

## Core Principle

After the geometry is reduced, eccentricity should often be read, not re-solved.

Typical readout forms:

- `e = c / a`
- special triangle ratios
- slope / asymptote relations
- directly controlled focal-distance ratios

## Good Signals

- a 3-4-5 closure appears
- a 1 : 2 : √5 or similar remembered ratio appears
- orientation and control triangle are already fixed

## Hard Rule

Only call it a direct readout if the control triangle or ratio is already visible.
If it still has to be built by long algebra, it is not a readout yet.
