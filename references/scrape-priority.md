## Low-Priority

Do not spend much more time broadly scraping:

- generic biography pages
- broad search for `伍博士` without method keywords
- already well-covered `七个三角形`
- already well-covered `尺子量法 / 比大小`

## Decision Rule

If a new public source:

- adds exact list contents
- adds a new direct exemplar
- clarifies how a sparse family actually works

then it is high-value.

If it only repeats branding already seen, it is low-value.
