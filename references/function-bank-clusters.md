# Wu Boshi Function-Bank Clusters

This file records the recurring “function bank / mother function / over-standard function” idea visible in public series.

## Core Principle

The public teaching posture repeatedly suggests:

- do not treat each function as new
- route it into a known behavior bank

That bank may be organized by:

- shape
- symmetry
- translation / scaling
- asymptotic behavior
- hidden normal form

## Public Cues

- `高考常考的13个超纲函数`
- `高考数学常考十大同构母函数`
- `六大完美超越函数`

## Extracted Method Meaning

### 1. Behavior Bank

What to ask:

- what does this function act like?
- which known behavior family should own it?

### 2. Transformation Bank

What to ask:

- can this be rewritten as a translation, scaling, or deformation of a known mother object?

### 3. Comparison Bank

What to ask:

- what monotone axis, convexity shape, or asymptotic regime already controls it?

### 4. Lower-Floor Version

What to ask:

- what familiar graph, curve, or elementary mechanism is hiding under the advanced wrapper?

## Hard Rule

Do not keep a “function bank” as a static memory list only.

Each bank entry should eventually answer:

- recognition cue
- default primitive
- common bait route
- cheapest faithful seal
