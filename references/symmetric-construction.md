# Wu Boshi Symmetric Construction

This file defines the move:

- construct a difference or comparison object centered at a special point so that symmetry does the heavy work

## Core Principle

Many two-variable or two-point problems become one-variable once you construct:

- `F(x)=f(x0+x)-f(x0-x)`

or a similar centered object.

This turns:

- two-sided comparison

into:

- one symmetric monotonicity problem

## Common Uses

- proving offset inequalities around an extremum point
- comparing two points with equal function value
- converting “two roots / two points” geometry into one centered parameter

## Hard Rule

The chosen center must be structurally meaningful:

- extremum point
- symmetry center
- midpoint
- natural balance point

Otherwise the construction is cosmetic.
