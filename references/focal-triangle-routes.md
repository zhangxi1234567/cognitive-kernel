# Wu Boshi Focal Triangle Routes

This file defines the move:

- treat the focal triangle as the real control object in conic problems

## Core Principle

A conic problem may pretend to be about coordinates, but the real control often sits in:

- one focal triangle
- one focal quadrilateral
- one angle / side relation among focal distances

## Common Route Types

### 1. Definition Route

- use conic definition as side-length relation

### 2. Triangle-Type Route

- identify right / isosceles / 30-60-90 / 3-4-5 / golden triangle closures

### 3. Area / Cosine-Theorem Route

- when the focal triangle is not right, use the cosine theorem as the final seal

## Hard Rule

Do not default to coordinates before asking whether a focal triangle already carries the target.
