# Wu Boshi Knowledge Ingress

This file defines how the skill should make hard knowledge enter the mind in a low-resistance way.

This is not decorative pedagogy.
It should change the solve path.

## Core Rule

Do not present a new concept as a new creature if it is secretly an old creature in new clothes.

Prefer:

- old friend in new clothing
- low-level familiar system
- one clean action model

Examples:

- matrix -> equation system / transform machine
- derivative -> instant speed / local response
- integral -> accumulation
- probability distribution -> weighted population
- chemical equilibrium -> two-sided tug toward a stable balance

## Operational Use

When a new concept appears, ask:

1. what old thing does this behave like?
2. what move does that old thing suggest?
3. what part of the original problem becomes cheaper after the translation?

If the answer changes only the explanation but not the next step, the ingress is decorative, not operational.

## Success Condition

The concept enters the mind correctly if:

- the learner can act on it
- the next solving move becomes clearer
- the original formal meaning can still be mapped back
