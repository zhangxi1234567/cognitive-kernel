# Route Memory Boundary

This file no longer defines route-memory shape.

Its only active claim is:

- later behavior may be sharpened by prior runs
- without preserving replayable route advice

## Forbidden Memory Forms

Do not preserve memory as:

- route logs
- route rankings
- rejected route sets
- future-route hints
- composition advice
- comparative route tables

## Final Rule

Memory may sharpen later competition.
It must not become delayed choreography.
