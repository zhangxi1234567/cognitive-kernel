## Boundary

Public kernel does not mean fixed order.

It does not mean:

- one primitive per domain
- one primitive per task type
- one fixed solve ladder
- or a public tactic menu

It only means the live runtime should present the smallest generic families first,
and let more specialized direct families stay subordinate to them.
